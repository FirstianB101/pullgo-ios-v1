# pullgo-ios

자체 홈페이지가 없는 소규모 학원들을 위한 학원 관리 플랫폼, 풀고의 iOS 프로젝트입니다.

이 프로젝트는 서버, 웹, 안드로이드, iOS로 각각 개발되는 중입니다.
